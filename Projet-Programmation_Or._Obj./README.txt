Pour lancer le jeu, exécutez le fichier Main.java présent dans le dossier Jeu.
Celui-ci ne comporte qu'une fonction public static void main(String[] args)
Dans la partie graphique, naviguez dans les menus avec zqsd
Appuyez sur enter quand votre curseur est à l'endroit où vous voulez poser votre tour puis choisissez la tour que vous voulez
